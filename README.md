# premium-insurance-calculator
A web-based insurance premium calculator with login system, risk analysis, and statistics dashboard.

## Features
- 🔐 User Login & Registration
- 📊 Premium Calculation with Risk Analysis
- 🗄️ SQLite Database Integration
- 📈 Statistics Dashboard with Charts
- 🎨 Modern Glassmorphism UI

## Tech Stack
- Python (Flask)
- HTML, CSS
- SQLite
- Chart.js

## How to Run
1. Install dependencies
2. Run `python app.py`
3. Open `http://127.0.0.1:5000`
